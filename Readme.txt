项目仓库地址：https://github.com/chiyu-git/chiyu-design

运行本项目需要按如下步骤安装必须的环境：

\1.   安装Node环境，检索NodeJS进入官网下载最新的NodeJS安装包，然后进行安装即可。

\2.   安装MySQL，去MySQL官网下载社区版本的安装包，按照流程完成MySQL的安装即可。

\3.   通过GitHub仓库下载本项目的压缩文件或者克隆到本地，在candidate，interviewer和server文件夹下分别执行 npm install 命令，即可安装各个客户端的所有依赖。

\4.   在server文件夹下执行npm start命令，可以在本地开启服务器。在candidate和interviewer文件夹下执行npm start命令可以开启相应的客户端。
